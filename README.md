# shopping-cart
Shopping Cart Demo Application


## Setup

docker-compose build

docker-compose run --rm app ./node_modules/.bin/sequelize-cli db:migrate

#### Products test
docker-compose run --rm app ./node_modules/.bin/sequelize-cli db:seed:all

## Run Server

docker-compose up
